1. **Создан файл CHANGELOG.md** — для отслеживания истории значимых изменений в проекте. Добавлена информация о последних 4 версиях, включая текущую версию 2.3.0 с улучшениями UX.

3. **Создан скрипт публикации изменений** — `publish_ux_changes.sh` для автоматизации процесса публикации изменений UX в репозиторий:
   - Добавляет все новые и измененные файлы
   - Создает коммит с детальным описанием изменений
   - Содержит инструкцию для отправки изменений в репозиторий

## Размещение изменений в репозитории

Для публикации всех изменений в репозиторий достаточно выполнить:

```bash
./publish_ux_changes.sh
git push origin main
```

Эти команды добавят все созданные файлы, создадут коммит с подробным описанием изменений и опубликуют их в основную ветку репозитория.
