# План улучшений для InProgressTasksList

## Требования к компоненту

Компонент `InProgressTasksList` должен обеспечивать следующую функциональность:

1. Отображение задач, находящихся в процессе выполнения
2. Сортировка задач по приоритету, дедлайну и исполнителю
3. Фильтрация задач по приоритету
4. Возможность сворачивать/разворачивать список задач
5. Статистика по количеству задач в разрезе приоритетов
6. Визуальная индикация прогресса выполнения каждой задачи
7. Адаптивность для различных размеров экрана
8. Визуальное выделение срочных задач и задач с истекающим дедлайном

## Проблемы текущей реализации

1. Отсутствие фильтрации и сортировки задач
2. Отсутствие статистики по задачам
3. Ограниченная визуализация прогресса задач
4. Отсутствие возможности сворачивать список задач
5. Отсутствие выделения срочных задач
