# Архитектура проекта OnboardPro

## Общий обзор

OnboardPro - это платформа для автоматизации процесса онбординга новых сотрудников. Проект построен с использованием современных технологий и следует принципам REST архитектуры.

## Технический стек

### Backend

- Python 3.11
- Django 5.2
- Django REST Framework
- djangorestframework-simplejwt
- PostgreSQL 15
- Docker & Docker Compose

### Frontend

- HTML5
- TailwindCSS
- JavaScript (Vanilla JS)
- Локальное хранилище для JWT токенов

## Архитектура безопасности

### Аутентификация

- JWT (JSON Web Tokens) для безопасной аутентификации
- Два типа токенов:
  - Access Token: короткоживущий токен для доступа к API
  - Refresh Token: долгоживущий токен для обновления Access Token
- Клиентская часть:
  - Форма входа с валидацией
  - Хранение токенов в localStorage
  - Автоматическая переадресация после успешной аутентификации

### Авторизация

Система ролевой авторизации с тремя уровнями доступа:

- Employee: базовый доступ
- Manager: расширенный доступ для руководителей
- HR: полный доступ к управлению персоналом

Реализована через:

- Кастомную модель User с полем role
- RoleBasedPermission для проверки прав доступа
- Декларативное указание required_role в views

## Модели данных

### User

Расширенная модель пользователя Django с дополнительными полями:

- role: Роль пользователя (employee, manager, hr)
- department: Отдел пользователя
- Стандартные поля Django User (username, email, password, etc.)
