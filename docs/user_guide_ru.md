## Введение

OnboardPro — это веб-приложение, разработанное для оптимизации процесса адаптации сотрудников. Оно предоставляет инструменты для HR-менеджеров, руководителей отделов и сотрудников для управления задачами, предоставления обратной связи и отслеживания прогресса.

## Начало работы

1. Откройте приложение по адресу http://localhost:3000
2. На странице входа вы увидите современный интерфейс с описанием ключевых особенностей системы в левой части экрана
3. Введите ваш email и пароль, которые вам предоставил системный администратор или HR-менеджер
4. При необходимости можете отметить опцию "Запомнить меня" для сохранения сессии
5. Нажмите кнопку "Войти"
6. После успешного входа вы будете автоматически перенаправлены на страницу дашборда в соответствии с вашей ролью:
   - HR: HR Dashboard
   - Менеджер: Manager Dashboard
   - Сотрудник: Dashboard

## Роли пользователей

- **HR**: Полный доступ ко всем функциям, включая создание, редактирование и удаление планов адаптации
- **Менеджер**: Доступ к Dashboard, Manager Dashboard (только просмотр планов адаптации), Feedback, Profiles
- **Сотрудник**: Доступ к Dashboard, Feedback

## Безопасность и доступ к страницам

- Неавторизованные пользователи видят только страницу логина (обновление от 30.04.2025)
- Защита страниц в соответствии с ролевой моделью (обновление от 30.04.2025):
  - Пользователи могут получить доступ только к страницам, соответствующим их роли
  - При попытке доступа к запрещенной странице происходит автоматическое перенаправление
  - Автоматическая проверка авторизации при каждом переходе между страницами
- Перенаправление на страницу логина при истечении срока действия токена (обновление от 30.04.2025)

## Функции

- **Dashboard**: Просмотр и управление вашими задачами
- **Manager Dashboard**: Управление задачами команды и сотрудниками
- **HR Dashboard**: Доступ к HR-метрикам и аналитике
- **Feedback**: Предоставление и получение обратной связи
- **Profiles**: Управление профилями пользователей
- **Integrations**: Подключение к внешним сервисам

## Работа с задачами

### Просмотр задач

1. Перейдите на страницу Dashboard
2. Вы увидите список задач, назначенных вам
3. Задачи теперь имеют улучшенную визуализацию приоритетов (обновление от 30.04.2025):
   - Задачи с высоким приоритетом имеют заголовок с красноватым фоном
   - Задачи со средним приоритетом имеют заголовок с желтоватым фоном
   - Задачи с низким приоритетом имеют заголовок с зеленоватым фоном
4. Используйте фильтры для отображения задач по статусу и приоритету

### Создание задачи (для HR и менеджеров)

1. На странице Manager Dashboard нажмите кнопку "Добавить задачу"
2. Заполните форму создания задачи:
   - Название
   - Описание
   - Сотрудник (кому назначается задача)
   - Приоритет (Высокий, Средний, Низкий)
   - Срок выполнения
3. Нажмите "Сохранить"

### Обновление статуса задачи

1. Найдите задачу в списке
2. Используйте выпадающий список статусов для изменения статуса
3. Изменения сохранятся автоматически

### Редактирование задачи (для пользователей с ролью HR, добавлено 29.04.2025)

1. На странице Manager Dashboard найдите задачу для редактирования
2. Нажмите на кнопку "Редактировать" (значок карандаша)
3. В появившемся модальном окне вы можете изменить:
   - Название задачи
   - Описание
   - Назначенного сотрудника
   - План адаптации
   - Приоритет
   - Срок выполнения
4. Нажмите "Сохранить изменения"

## Работа с календарём

### Просмотр задач в календаре

1. Перейдите на страницу "Календарь" в главном меню
2. Задачи отображаются в виде событий на календаре
3. Цвет события соответствует приоритету задачи:
   - Красный — высокий приоритет
   - Жёлтый — средний приоритет
   - Зелёный — низкий приоритет

### Изменение даты задачи с помощью перетаскивания (обновлено 29.04.2025)

1. Наведите курсор на задачу в календаре
2. Нажмите и удерживайте левую кнопку мыши
3. Перетащите задачу на новую дату в календаре
4. Отпустите кнопку мыши — задача будет автоматически перемещена на новую дату
5. Система отобразит уведомление об успешном обновлении даты задачи

### Изменение представления календаря

1. Используйте кнопки в верхнем правом углу календаря для выбора представления:
   - Месяц — просмотр задач на месяц
   - Неделя — просмотр задач на неделю
   - День — просмотр задач на день
2. Используйте стрелки для перехода к предыдущему/следующему периоду
3. Кнопка "Сегодня" позволяет быстро вернуться к текущей дате

### Фильтрация задач в календаре

1. Используйте выпадающее меню фильтра в верхней части календаря
2. Вы можете фильтровать задачи по:
   - Приоритету
   - Статусу
   - Плану адаптации
   - Назначенному сотруднику (для HR и менеджеров)

## Работа с планами адаптации

### Просмотр планов (для HR и менеджеров)

1. Перейдите на страницу Manager Dashboard или HR Dashboard
2. Прокрутите страницу до раздела "Управление планами адаптации"
3. Вы увидите список существующих планов адаптации и их основную информацию:
   - Название плана
   - Целевая роль (сотрудник, менеджер)
   - Описание (если есть)
   - Назначенный сотрудник (если план уже назначен)
   - Прогресс выполнения задач в виде индикатора

### Просмотр шаблонов задач (только для HR, обновление от 06.05.2025)

1. Перейдите на страницу Manager Dashboard
2. Прокрутите страницу до раздела "Управление планами адаптации"
3. Нажмите на блок "Шаблоны задач", чтобы развернуть список шаблонов
4. В списке шаблонов отображается следующая информация:
   - Название шаблона
   - Описание
   - Роль
   - Отдел
5. Вы можете свернуть список шаблонов, нажав на блок "Шаблоны задач" повторно

### Создание плана адаптации (только для HR, обновление от 06.05.2025)

1. Перейдите на страницу Manager Dashboard
2. В разделе "Управление планами адаптации" нажмите кнопку "Создать план"
3. В появившемся модальном окне заполните форму:
   - Название плана
   - Описание (опционально)
   - Целевая роль (сотрудник, менеджер, HR)
   - Выберите сотрудника, для которого создается план (опционально)
   - Добавьте шаблонные задачи из списка доступных шаблонов
   - Добавьте кастомные задачи при необходимости
   - При желании включите интеграцию с Google Calendar
4. Нажмите "Создать план"
5. Созданный план появится в списке планов адаптации

### Создание шаблона задачи (только для HR, обновление от 06.05.2025)

1. Перейдите на страницу Manager Dashboard
2. В разделе "Управление планами адаптации" нажмите кнопку "Создать шаблон"
3. В появившемся модальном окне заполните форму:
   - Название шаблона\*
   - Описание (опционально)
   - Приоритет (низкий, средний, высокий)
   - Длительность (в днях)\*
   - Целевая роль (сотрудник, менеджер, HR)
   - Отдел (опционально)
4. Нажмите "Создать шаблон"
5. Созданный шаблон появится в списке шаблонов задач

### Просмотр карточки плана адаптации (обновление от 12.05.2025)

Карточки планов адаптации теперь содержат следующую информацию:
1. Фотография сотрудника и его полное имя (имя и фамилия) в заголовке карточки
2. Если у сотрудника нет фотографии, отображается стандартная иконка профиля
3. Бейдж с ролью ("Менеджер" или "Сотрудник") под именем
4. Описание плана (если есть)
5. Индикатор прогресса выполнения задач
6. Количество выполненных задач из общего числа

### Редактирование плана адаптации (только для HR, обновление от 06.05.2025)

1. Найдите нужный план в разделе "Управление планами адаптации"
2. Нажмите на значок редактирования (карандаш) в правом верхнем углу карточки плана
3. В открывшемся модальном окне внесите необходимые изменения:
   - Измените название, описание или целевую роль плана
   - Добавьте или удалите шаблонные задачи
   - Добавьте или удалите кастомные задачи
   - Измените назначенного сотрудника (если требуется)
   - Настройте интеграцию с Google Calendar
4. Нажмите "Сохранить изменения"

### Редактирование шаблона задачи (только для HR, обновление от 06.05.2025)

1. Разверните список шаблонов задач в разделе "Управление планами адаптации"
2. Найдите нужный шаблон в списке
3. Нажмите на значок редактирования (карандаш) в правой части строки
4. В открывшемся модальном окне внесите необходимые изменения:
   - Измените название, описание, приоритет или длительность
   - Измените целевую роль или отдел
5. Нажмите "Обновить шаблон"

### Удаление плана адаптации (только для HR, обновление от 06.05.2025)

1. Найдите нужный план в разделе "Управление планами адаптации"
2. Нажмите на значок удаления (корзина) в правом верхнем углу карточки плана
3. В появившемся диалоге подтверждения нажмите "Удалить"
4. Обратите внимание: нельзя удалить план с активными задачами

### Удаление шаблона задачи (только для HR, обновление от 06.05.2025)

1. Разверните список шаблонов задач в разделе "Управление планами адаптации"
2. Найдите нужный шаблон в списке
3. Нажмите на значок удаления (корзина) в правой части строки
4. В появившемся диалоге подтверждения нажмите "Удалить"

### Назначение плана сотруднику

1. Перейдите в профиль сотрудника
2. В разделе "Планы адаптации" выберите план из выпадающего списка
3. Нажмите "Назначить"

## Устранение неполадок

### Проблемы с авторизацией и доступом к страницам (обновление от 30.04.2025)

- **Перенаправление на страницу логина**:

  1. Ваша сессия могла истечь - выполните повторный вход в систему
  2. Проверьте правильность введенных учетных данных
  3. Убедитесь, что ваша роль позволяет получить доступ к запрашиваемой странице
  4. При необходимости очистите кэш браузера и файлы cookie

- **Ошибка "Доступ запрещен" при попытке открыть страницу**:
  1. Убедитесь, что ваша учетная запись имеет необходимые права доступа
  2. Возможно, страница доступна только для пользователей с определенной ролью
  3. Обратитесь к HR-менеджеру для уточнения ваших прав доступа

### Проблемы с отображением графиков и визуализаций (обновление от 30.04.2025)

- **Графики отображаются некорректно или не отображаются**:
  1. Убедитесь, что в вашем браузере включен JavaScript
  2. Попробуйте обновить страницу (F5)
  3. Очистите кэш браузера и перезагрузите страницу
  4. Попробуйте использовать другой современный браузер (Chrome, Firefox, Safari, Edge)

### Проблемы с функциональностью календаря

- **Не удается перетащить задачу в календаре** (исправлено 29.04.2025):

  1. Убедитесь, что вы вошли в систему с правильными учетными данными
  2. Проверьте, достаточно ли у вас прав для изменения задачи (обычно требуется роль HR или менеджера)
  3. Обновите страницу календаря для синхронизации с сервером
  4. Если проблема сохраняется, обратитесь к администратору системы

- **Изменение даты задачи через перетаскивание не сохраняется**:

  1. Убедитесь, что у вас стабильное интернет-соединение
  2. Проверьте, что задача не находится в состоянии "Завершено" - такие задачи могут быть заблокированы для изменений
  3. Очистите кэш браузера и попробуйте снова
  4. Если проблема сохраняется, попробуйте изменить дату задачи через форму редактирования

### Другие распространенные проблемы

- Если у вас возникли проблемы с входом, убедитесь, что вы используете правильные учетные данные
- При проблемах с отображением интерфейса, попробуйте очистить кэш браузера или использовать другой браузер
- Если задача не отображается в вашем списке, проверьте фильтры и убедитесь, что она была вам назначена
- Для технической поддержки обратитесь к системному администратору или напишите на support@onboardpro.com
