## Введение

OnboardPro — это веб-приложение, разработанное для оптимизации процесса адаптации сотрудников. Оно предоставляет инструменты для HR-менеджеров, руководителей отделов и сотрудников для управления задачами, предоставления обратной связи и отслеживания прогресса.

## Начало работы

1. Откройте приложение по адресу http://localhost:3000
2. Войдите с вашими учетными данными:
   - HR: test@onboardpro.com / test123
   - Менеджер: manager@onboardpro.com / test123
   - Сотрудник: employee@onboardpro.com / test123

## Роли пользователей

- **HR**: Полный доступ ко всем функциям
- **Менеджер**: Доступ к Dashboard, Manager Dashboard, Feedback, Profiles
- **Сотрудник**: Доступ к Dashboard, Feedback

## Функции

- **Dashboard**: Просмотр и управление вашими задачами
- **Manager Dashboard**: Управление задачами команды и сотрудниками
- **HR Dashboard**: Доступ к HR-метрикам и аналитике
- **Feedback**: Предоставление и получение обратной связи
- **Profiles**: Управление профилями пользователей
- **Integrations**: Подключение к внешним сервисам

## Улучшенный интерфейс

- Фиксированный футер с копирайтом "© 2025 magna_mentes".
- Адаптивная навигация: иконки+текст (десктоп ≥1280px), только иконки (<1280px), бургер-меню (<768px), поддержка сверхшироких экранов (≥1920px).
- Информация о текущем пользователе (email, роль: HR, Менеджер, Сотрудник) отображается в шапке (≥1280px).
- Фронтенд перенесен на React для улучшения интерактивности и поддерживаемости.
- Интерфейс полностью адаптирован под мобильные и планшетные устройства.

## Работа с задачами

### Просмотр задач

1. Перейдите на страницу Dashboard
2. Вы увидите список задач, назначенных вам
3. Используйте фильтры для отображения задач по статусу и приоритету

### Создание задачи (для HR и менеджеров)

1. На странице Manager Dashboard нажмите кнопку "Добавить задачу"
2. Заполните форму создания задачи:
   - Название
   - Описание
   - Сотрудник (кому назначается задача)
   - Приоритет (Высокий, Средний, Низкий)
   - Срок выполнения
3. Нажмите "Сохранить"

### Обновление статуса задачи

1. Найдите задачу в списке
2. Используйте выпадающий список статусов для изменения статуса
3. Изменения сохранятся автоматически

## Работа с планами адаптации

### Просмотр планов (для HR и менеджеров)

1. Перейдите на страницу Manager Dashboard или HR Dashboard
2. Перейдите на вкладку "Планы адаптации"
3. Вы увидите список существующих планов

### Создание плана адаптации (для HR и менеджеров)

1. На странице планов адаптации нажмите "Создать план"
2. Заполните форму:
   - Название плана
   - Описание
   - Целевая роль (сотрудник, менеджер)
3. Добавьте задачи к плану, нажав "Добавить задачу" в разделе задач плана
4. Нажмите "Сохранить"

### Назначение плана сотруднику

1. Перейдите в профиль сотрудника
2. В разделе "Планы адаптации" выберите план из выпадающего списка
3. Нажмите "Назначить"

## Обратная связь

### Просмотр обратной связи

1. Перейдите на страницу Feedback
2. Вы увидите список полученной и отправленной обратной связи

### Отправка обратной связи

1. На странице Feedback нажмите "Написать отзыв"
2. Выберите получателя из выпадающего списка
3. Введите текст обратной связи
4. Нажмите "Отправить"

## Интеграции

### Telegram

- Получайте уведомления о новых задачах и обратной связи в Telegram
- Для настройки, обратитесь к HR-менеджеру для получения ссылки на бота

### Google Calendar

- События и сроки выполнения задач автоматически добавляются в ваш Google Calendar
- Для настройки, перейдите в раздел Integrations и следуйте инструкциям

### Workable

- HR-менеджеры могут импортировать новых сотрудников из Workable
- Для настройки, перейдите в раздел Integrations (доступно только для HR)

## Работа с профилями пользователей

### Доступ к странице профилей

- HR-менеджеры имеют полный доступ ко всем функциям управления пользователями
- Менеджеры могут просматривать профили всех пользователей

### Просмотр и фильтрация пользователей

1. Перейдите на страницу Profiles
2. Пользователи автоматически группируются по ролям (HR, Менеджеры, Сотрудники)
3. Используйте фильтры в верхней части страницы для отображения:
   - Всех пользователей
   - Только активных пользователей
   - Только заблокированных пользователей
4. Воспользуйтесь полем поиска для быстрого нахождения пользователя по email или отделу

### Управление пользователями (только для HR)

#### Создание нового пользователя

1. Нажмите кнопку "Добавить пользователя" в верхней части страницы
2. Заполните форму:
   - Email
   - Пароль
   - Роль (HR, Менеджер, Сотрудник)
   - Отдел
3. Нажмите "Создать"

#### Редактирование пользователя

1. Найдите нужного пользователя в списке
2. Нажмите кнопку редактирования (значок карандаша)
3. Внесите необходимые изменения
4. Нажмите "Сохранить"

#### Блокировка/разблокировка пользователя

1. Найдите нужного пользователя в списке
2. Нажмите кнопку блокировки/разблокировки (значок замка)
3. Подтвердите действие

#### Сброс пароля пользователя

1. Найдите нужного пользователя в списке
2. Нажмите кнопку сброса пароля (значок ключа)
3. Система отобразит временный пароль в модальном окне
4. Скопируйте или запишите этот пароль и безопасно передайте его пользователю

#### Удаление пользователя

1. Найдите нужного пользователя в списке
2. Нажмите кнопку удаления (значок корзины)
3. Подтвердите удаление

## Устранение неполадок

- Если у вас возникли проблемы с входом, убедитесь, что вы используете правильные учетные данные
- При проблемах с отображением интерфейса, попробуйте очистить кэш браузера или использовать другой браузер
- Если задача не отображается в вашем списке, проверьте фильтры и убедитесь, что она была вам назначена
- Для технической поддержки обратитесь к системному администратору или напишите на support@onboardpro.com
