# Безопасность OnboardPro

## Аутентификация

### JWT (JSON Web Tokens)

В OnboardPro используются JWT токены для аутентификации пользователей. JWT токены генерируются с использованием алгоритма HS256 и содержат следующую информацию:

- sub: email пользователя
- exp: время истечения токена

### Хранение паролей

Пароли пользователей хранятся в базе данных в хешированном виде с использованием bcrypt. Это обеспечивает:

- Защиту от атак перебором
- Уникальную соль для каждого пароля
- Невозможность восстановления оригинального пароля

## Настройки безопасности

### Переменные окружения

Критически важные данные хранятся в файле `.env`:

- DATABASE_URL: URL подключения к базе данных
- SECRET_KEY: секретный ключ для генерации JWT токенов

**Важно**: Никогда не включайте файл `.env` в систему контроля версий.

### CORS

Настройки CORS в FastAPI позволяют:

- Контролировать доступ к API
- Указывать разрешенные источники
- Управлять разрешенными методами и заголовками

## Рекомендации

1. Регулярно обновляйте зависимости
2. Используйте HTTPS в продакшене
3. Настройте rate limiting
4. Проводите регулярный аудит безопасности

## Роли пользователей

### Employee

- Базовый доступ к системе
- Просмотр своего профиля
- Выполнение задач онбординга

### Manager

- Просмотр профилей сотрудников своего отдела
- Управление задачами онбординга
- Отслеживание прогресса

### HR

- Полный доступ к системе
- Создание и управление пользователями
- Настройка процессов онбординга
