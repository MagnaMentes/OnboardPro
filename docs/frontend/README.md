# 🎨 Фронтенд OnboardPro

## 🛠 Технологии

- 📄 HTML5
- 💅 TailwindCSS для стилизации с PostCSS 8
- ⚡️ React для интерактивности
- 🔤 Inter font для типографики
- 📱 Адаптивный дизайн для всех устройств

## ⚙️ Настройка и установка

1. 📥 Установите Node.js и npm
2. 🔧 Выполните:
   ```bash
   npm install --legacy-peer-deps
   ```
3. 🚀 Для разработки запустите:
   ```bash
   npm start
   ```
4. 📦 Для сборки:
   ```bash
   npm run build
   ```

### 🪝 Правила использования React Hooks

- 🔄 Хуки должны вызываться только на верхнем уровне функциональных компонентов или в кастомных хуках
- ❌ Запрещено вызывать хуки внутри условий, циклов или вложенных функций
- 🧩 Для модальных окон следуйте паттерну:
  ```javascript
  const ModalComponent = () => {
    // 1. Хуки всегда в начале компонента
    const [localState, setLocalState] = useState(initialValue);
    
    // 2. Синхронизация с внешним состоянием
    useEffect(() => {
      if (isModalOpen) {
        setLocalState(externalData);
      }
    }, [isModalOpen, externalData]);
    
    // 3. Условный рендер только после всех хуков
    if (!isModalOpen) return null;
    
    return (
      // JSX модального окна
    );
  };
  ```
- 📑 Используйте ESLint с плагином react-hooks для автоматического обнаружения ошибок
