# 🎨 Фронтенд OnboardPro

## 🛠 Технологии

- 📄 HTML5
- 💅 TailwindCSS для стилизации
- ⚡️ Vanilla JavaScript для интерактивности
- 🔤 Inter font для типографики

## 📁 Структура проекта

```
frontend/
├── src/                # 📂 Исходный код
│   ├── layouts/        # 🏗 Шаблоны страниц
│   │   └── base.html   # 📑 Базовый шаблон
│   ├── input.css       # 🎨 Основной файл стилей
│   ├── login.html      # 🔑 Страница входа
│   └── login.js        # 🔐 Логика авторизации
├── dist/               # 📦 Скомпилированные файлы
│   └── output.css      # 🎭 Скомпилированные стили
├── package.json        # 📋 Зависимости и скрипты
└── tailwind.config.js  # ⚙️ Конфигурация TailwindCSS
```

## ⚙️ Настройка и установка

1. 📥 Установите Node.js и npm
2. 🔧 Выполните:
   ```bash
   npm install
   ```
3. 🚀 Для разработки запустите:
   ```bash
   npm run watch
   ```
4. 📦 Для сборки:
   ```bash
   npm run build
   ```

## 🎨 Стилизация

### 🎯 Цветовая палитра

```javascript
colors: {
  primary: '#1D4ED8',    // 🔵 Основной цвет
  secondary: '#6B7280',  // ⚪️ Вторичный цвет
  accent: '#10B981',     // 💚 Акцентный цвет
  background: '#F3F4F6', // ⚪️ Фон
}
```

### 🔤 Типографика

- 📝 Основной шрифт: Inter
- 📜 Варианты начертания: 400 (regular), 600 (semibold), 700 (bold)

## 🧱 Компоненты

### 📑 Базовый шаблон (base.html)

Содержит:

- 🗂 Header с навигацией
- 📄 Основной контейнер для контента
- 📝 Footer с копирайтом

### 🔑 Форма входа (login.html)

- 📱 Адаптивный дизайн
- ✅ Валидация на стороне клиента
- ⚠️ Обработка ошибок

## ⚡️ JavaScript

### 🔐 Аутентификация (login.js)

- 📤 Отправка формы через fetch API
- 💾 Сохранение JWT в localStorage
- 🔄 Обработка ошибок и перенаправление

## 📋 Рекомендации по разработке

1. 🎯 Следуйте методологии Utility-First CSS от TailwindCSS
2. 📝 Используйте семантическую разметку HTML
3. 📱 Придерживайтесь принципа Mobile First
4. 🌐 Тестируйте в разных браузерах
