# Журнал изменений OnboardPro

## Версия 2.4.3 - 14.05.2025

### Улучшения

- Улучшен пользовательский интерфейс управления задачами:
  - Добавлена иконка инструментов в заголовок колонки действий в таблице "Управление задачами" вместо скрытого текста
  - Добавлен запрос на подтверждение при удалении задачи для предотвращения случайного удаления
  - Реализовано единое поведение подтверждения удаления во всех представлениях интерфейса

### Документация

- Обновлена техническая документация с описанием новых улучшений UI
- Обновлено руководство пользователя с описанием нового процесса удаления задач
- Добавлена запись в базе знаний о внесенных улучшениях UI и предотвращении случайного удаления задач

## Версия 2.4.2 - 14.05.2025

### Исправления

- Исправлена структура JSX в разделе шаблонов задач на панели менеджера (ManagerDashboard.jsx)
- Реализованы плавные анимации для панелей фильтров и шаблонов задач
- Устранено дублирование кода отображения шаблонов задач

### Документация

- Добавлена информация об исправлении анимаций в базу знаний проекта

## Версия 2.4.1 - 13.05.2025

### Улучшения

- Улучшен пользовательский интерфейс системы навигации:
  - Изменена ориентация элементов навигации для выравнивания по правому краю
  - Стандартизировано использование компонента `<Link>` для навигационных элементов
  - Унифицирована стилизация элементов навигации во всех видах представления

### Исправлено

- Устранены проблемы с отображением навигационных элементов на двух строках
- Убраны внутренние отступы между иконками и текстом последнего пункта меню

## Версия 2.3.2 - 13.05.2025

### Исправлено

- Исправлена ошибка в обработчике WebSocket для аналитики: `TypeError: setHasRealtimeUpdates is not a function`, которая появлялась в консоли браузера
- Добавлена дополнительная проверка типа для функции обратного вызова в модуле `hrWebSocketHelpers.js`

## Версия 2.3.1 - 12.05.2025

### Изменено

- Улучшен интерфейс карточек планов адаптации в панели менеджера:
  - Имя и фамилия сотрудника теперь отображаются в заголовке карточки вместо названия плана
  - Добавлено отображение фото сотрудника рядом с именем
  - Сохранен бейдж с ролью ("Сотрудник" или "Менеджер") под именем
  - Поддерживаются стандартные иконки-заглушки для сотрудников без фото

### Улучшено

- Повышена наглядность панели менеджера для быстрой идентификации сотрудников и их планов адаптации

## Версия 2.3.0 - 09.05.2025

### Добавлено

- Полностью обновленный интерфейс страницы входа с адаптивным дизайном

- Исправлена проблема с отображением сообщения об ошибке авторизации

## Версия 1.8.0 - 28.04.2025

### Добавлено

- Расширенная фильтрация задач по статусу, приоритету, сотруднику и плану
- Поиск задач по названию и описанию
- Сортировка задач по названию, приоритету, сроку и статусу
- Возможность обновления списка задач из базы данных
- Счетчик отображаемых задач с информацией о примененных фильтрах
- Создание планов через модальные формы (только для HR)
- Редактирование существующих планов (только для HR)
- Удаление планов без связанных задач (только для HR)
- Проверка наличия связанных задач перед удалением плана
- Возможность обновления списка планов из базы данных
- Визуальные уведомления об успешных операциях с планами
- Контроль прав доступа на основе роли пользователя

### Изменено

- В заголовке карточки пользователей отображается ФИО вместо email
- В теле карточки пользователей отображаются только телефон, email и отдел
- Упрощена шапка интерфейса без отображения информации о пользователе

### Исправлено

- Исправлена проблема отображения фотографий пользователей
- Исправлены проблемы с URL для изображений и API запросов
