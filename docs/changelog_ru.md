# Журнал изменений OnboardPro

## Версия 2.3.0 - 09.05.2025

### Добавлено
- Полностью обновленный интерфейс страницы входа с адаптивным дизайном
- Левая информационная панель на странице входа с ключевыми особенностями системы
- Отзывы клиентов в информационной панели
- Функция "Запомнить меня" на странице входа
- Ссылки в футере на страницы "Правила", "Конфиденциальность" и "Помощь"

### Изменено
- Улучшен визуальный дизайн полей ввода на странице входа
- Доработана валидация формы входа с более информативными сообщениями об ошибках

### Исправлено
- Исправлена проблема с отображением сообщения об ошибке авторизации

## Версия 2.2.0 - 02.05.2025

### Добавлено
- Улучшенная синхронизация фактического списка задач и аналитических данных

### Изменено
- Усовершенствован интерфейс HR Dashboard
- Удалено дублирующее табличное представление блока "Задачи в процессе"

### Исправлено
- Исправлена проблема с обнулением счетчика "Задачи в процессе" после перезагрузки страницы

## Версия 2.1.0 - 01.05.2025

### Добавлено
- Механизм управления кэшированием данных аналитики
- Улучшенная обработка WebSocket событий
- Замена иконки CircleStackIcon на QueueListIcon для статуса "В очереди"

### Изменено
- Усовершенствован механизм обхода кэша для получения актуальных данных

## Версия 2.0.0 - 30.04.2025

### Добавлено
- Компонент ProtectedRoute для проверки авторизации перед отображением страницы
- Функция validateToken для проверки валидности токена
- Улучшенный дизайн карточек задач с цветовой индикацией приоритета
- Обновленный компонент AnalyticsChart

### Изменено
- Стандартизация дизайна всех страниц приложения
- Обновлена структура компонентов App.js и Layout.js
- Упрощена логика авторизации, фокус только на UI

## Версия 1.9.0 - 29.04.2025

### Добавлено
- Модальное окно редактирования задач с возможностью изменения всех полей
- Полное редактирование задач для пользователей с ролью HR

### Исправлено
- Исправлена ошибка при обновлении даты задачи через drag-and-drop в календаре

## Версия 1.8.0 - 28.04.2025

### Добавлено
- Расширенная фильтрация задач по статусу, приоритету, сотруднику и плану
- Поиск задач по названию и описанию
- Сортировка задач по названию, приоритету, сроку и статусу
- Возможность обновления списка задач из базы данных
- Счетчик отображаемых задач с информацией о примененных фильтрах
- Создание планов через модальные формы (только для HR)
- Редактирование существующих планов (только для HR)
- Удаление планов без связанных задач (только для HR)
- Проверка наличия связанных задач перед удалением плана
- Возможность обновления списка планов из базы данных
- Визуальные уведомления об успешных операциях с планами
- Контроль прав доступа на основе роли пользователя

### Изменено
- В заголовке карточки пользователей отображается ФИО вместо email
- В теле карточки пользователей отображаются только телефон, email и отдел
- Упрощена шапка интерфейса без отображения информации о пользователе

### Исправлено
- Исправлена проблема отображения фотографий пользователей
- Исправлены проблемы с URL для изображений и API запросов
