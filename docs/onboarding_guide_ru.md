# Промт для senior fullstack-разработчика и UI/UX-дизайнера: быстрый вход в проект OnboardPro

## Обзор проекта

OnboardPro — веб-приложение для автоматизации и улучшения процесса адаптации новых сотрудников. Приложение позволяет HR-специалистам и менеджерам создавать планы адаптации, назначать и отслеживать выполнение задач, а сотрудникам — видеть свои задачи и отмечать их выполнение.

## Архитектура системы

- **Frontend**: React с Tailwind CSS, Heroicons
- **Backend**: Python FastAPI с SQLite и SQLAlchemy ORM
- **Инфраструктура**: Docker и docker-compose
- **Аутентификация**: JWT-токены
- **Внешние интеграции**: Telegram Bot, Google Calendar, Workable API

## Роли пользователей и модель доступа

Система поддерживает три уровня доступа:

1. **HR** — полный доступ к системе, управление пользователями, задачами и планами
2. **Менеджер** — доступ к просмотру профилей пользователей и полное управление задачами и планами адаптации
3. **Сотрудник** — ограниченный доступ к назначенным задачам и планам

## Ключевые функциональные модули

### Управление пользователями

- Создание/редактирование/удаление учетных записей (только HR)
- Просмотр профилей (HR и менеджеры)
- Блокировка/разблокировка пользователей (только HR)
- Сброс паролей (только HR)

### Управление задачами

- Создание и назначение задач сотрудникам
- Отслеживание статуса выполнения
- Организация задач по приоритетам и дедлайнам
- Связывание задач с планами адаптации

### Планы адаптации

- Создание типовых планов для разных ролей
- Настройка этапов и контрольных точек
- Автоматическое формирование задач из плана

### Обратная связь

- Сбор отзывов от сотрудников и менеджеров
- Анализ результатов адаптации

## Технические особенности и известные проблемы

1. **Проблема с авторизацией HR-пользователей**: При возникновении проблем с авторизацией требуется перезапуск контейнеров с помощью `docker-compose down && docker-compose up -d`.

2. **Дублирование данных**: В случае обнаружения дубликатов необходима полная пересборка с очисткой кэша:

```
docker-compose down
rm backend/onboardpro.db
docker-compose build --no-cache
docker-compose up -d
```

3. **Доступ к странице профилей**: Недавно был исправлен баг с доступом менеджеров к странице профилей — проверка в `Profiles.jsx` теперь учитывает обе роли:

```javascript
if (response.data.role !== "hr" && response.data.role !== "manager") {
  toast.error("У вас нет доступа к этой странице");
  navigate("/dashboard");
}
```

## UI/UX особенности

- **Адаптивный дизайн**:

  - Десктоп (≥1280px): Полная навигация с иконками и текстом
  - Планшет (≥768px, <1280px): Навигация только с иконками
  - Мобильный (<768px): Навигация через бургер-меню

- **Визуальная иерархия**:

  - Четкие заголовки разделов
  - Модальные окна для создания/редактирования задач и планов
  - Цветовое кодирование для разных ролей пользователей:
    - HR: пурпурный
    - Менеджер: синий
    - Сотрудник: зеленый

- **Компоненты**: Используется набор переиспользуемых UI-компонентов с единым стилем

## Как начать работу

1. **Клонирование и настройка**:

```bash
git clone [repo_url]
cd OnboardPro
```

2. **Настройка переменных окружения**:

   - Создайте файл `.env` с необходимыми переменными:
     - DATABASE_URL
     - SECRET_KEY
     - TELEGRAM_BOT_TOKEN
     - GOOGLE_CREDENTIALS_PATH
     - WORKABLE_API_KEY

3. **Запуск с Docker**:

```bash
docker-compose up -d
```

4. **Доступ к приложению**:
   - Frontend: http://localhost:3000
   - Backend API: http://localhost:8000
   - Документация API: http://localhost:8000/docs

## Структура кодовой базы

### Frontend

- `App.js` — главный компонент приложения
- `/components` — переиспользуемые UI-компоненты
- `/pages` — компоненты страниц (Dashboard, Profiles, etc.)
- `/context` — React Context провайдеры
- `/hooks` — пользовательские React хуки
- `/utils` — утилиты и хелперы

### Backend

- `main.py` — основной файл приложения FastAPI
- `models.py` — модели SQLAlchemy для БД
- `auth.py` — логика аутентификации и авторизации
- `database.py` — настройка соединения с БД
- `integrations.py` — интеграции с внешними сервисами
- `/migrations` — миграции Alembic
