# Журнал изменений проекта OnboardPro

Этот файл содержит историю значимых изменений в проекте.

## Версия 2.4.2 (14 мая 2025)

### Исправлено

- Исправлена структура JSX в разделе шаблонов задач на панели менеджера (ManagerDashboard.jsx)
- Реализованы плавные анимации для панелей фильтров и шаблонов задач
- Устранено дублирование кода отображения шаблонов задач

### Документация

- Добавлена информация об исправлении анимаций в базу знаний проекта

## Версия 2.4.1 (13 мая 2025)

### Улучшено

- Улучшен пользовательский интерфейс системы навигации:
  - Изменена ориентация элементов навигации для выравнивания по правому краю вместо центрирования
  - Стандартизировано использование компонента `<Link>` для навигационных элементов
  - Устранены избыточные отступы у последнего навигационного элемента
  - Унифицирована стилизация элементов навигации во всех видах представления (десктопное, планшетное, мобильное)

### Исправлено

- Устранены проблемы с отображением навигационных элементов на двух строках
- Убраны внутренние отступы между иконками и текстом последнего пункта меню

### Документация

- Обновлена техническая документация по компонентам навигации
- Добавлены подробные записи о внесенных изменениях в базу знаний проекта

## Версия 2.4.0 (13 мая 2025)

### Добавлено

- Создана модульная архитектура компонентов HR Dashboard:
  - `AnalyticsTabs.jsx`: компонент для переключения вкладок с поддержкой доступности
  - `StatCard.jsx`: переиспользуемый компонент для отображения статистики
  - `FilterPanels.jsx`: компоненты панелей фильтрации данных
  - `KPIPanel.jsx`: компонент для отображения ключевых показателей эффективности
  - `InProgressTasksList.jsx`: компонент для отображения задач в процессе выполнения
  - `hrAnalyticsHelpers.js`: утилиты для работы с данными аналитики
  - `hrWebSocketHelpers.js`: утилиты для работы с WebSocket соединениями

### Исправлено

- Устранена проблема с несоответствием счетчика задач в процессе и фактическим количеством задач в списке
- Исправлена обработка WebSocket соединений для стабильной работы в Docker-среде
- Оптимизированы ререндеринги компонентов страницы HR Dashboard
- Улучшена проверка и обработка входных данных компонентов

### Документация

- Добавлена техническая документация по компонентам HR Dashboard
- Обновлены README.md основного проекта и frontend части
- Создан документ с опытом рефакторинга для базы знаний проекта

## Версия 2.3.1 (12 мая 2025)

### Исправлено

- Исправлены пути API в компонентах фронтенда для корректной работы в Docker-окружении:
  - `PlanForm.jsx`, `TaskForm.jsx`, `DepartmentForm.jsx`, `Departments.jsx`, `Profiles.jsx`, `ManagerDashboard.jsx`, `TaskTemplateForm.jsx`, `HRDashboard.jsx`
- Удалены дублирующиеся префиксы `/api/` в URL-путях API
- Добавлено использование функции `getApiBaseUrl()` во всех компонентах для единообразия формирования URL
- Создана полная документация по исправлениям в файлах журнала проекта

## Версия 2.3.0 (12 мая 2025)

### Добавлено

- Создан модуль утилит `userUtils.js` для стандартизации отображения данных пользователей
- Добавлена функция `formatUserDisplayName` для форматирования имен пользователей
- Добавлена функция `getUserDisplayNameById` для получения имени пользователя по ID
- Добавлена функция `getUserInitials` для получения инициалов пользователя
- Добавлена документация по стандартам отображения пользовательских данных

### Исправлено

- В `TaskModal.jsx` исправлено отображение имен пользователей с использованием модуля утилит
- В `ManagerDashboard.jsx` заменена локальная функция `getUserDisplayNameById` на импорт из модуля утилит
- Исправлена ошибка с функцией `getUserEmailById`, вызывавшей сбой компиляции
- В `DepartmentForm.jsx` исправлено отображение менеджеров отделов

### Документация

- Создана подробная документация стандартов отображения данных пользователей (`user_display_standards.md`)
- Создан итоговый отчет о проведенных улучшениях UX (`user_display_summary_20250512.md`)
- Обновлен общий журнал изменений с информацией о выполненной работе
